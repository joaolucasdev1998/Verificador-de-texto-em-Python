Este projeto é um verificador de texto em python.
O programa solicita ao usuário um texto principal
e,em seguida,pede um segundo texto para ser procurado
dentro do primeiro.
Utiliza a estrutura condicional if/else e o operador in
para verificar se o conteúdo digitado está presente.
O objetivo do projeto é praticar entrada de dados,comparação de strings e lógica de programação em Python.
